## Bytecode VM interpreter for Lox

### Usage

Interpret a lox file
```sh
loxz <file.lox> 
```

Run in REPL mode

```sh
loxz
```

Flags:

| Flag         | Description                |
|--------------|----------------------------|
| `-h`, `--help` | Show help message          |
| `-t`, `--stack-tracing` | Enable VM stack tracing   |
| `-d`, `--debug`   | Enable debug output        |
